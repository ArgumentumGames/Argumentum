Intégration Html simple

Prérequis

La feuille de style et l'image suivantes doivent être insérée dans la page ou l'application web que vous utilisez:

  • rhetological_fallacies_francais.css
  • rhetological_fallacies_francaisFULL.png
Il vous suffit alors de copier et personaliser l'un des blocs suivants pour obtenir le même rendu.

Liste des fallacies:

  • Manipulations de l'esprit
  • Appel aux émotions
  • Déduction érronnée
  • Confusion entre causes et effets
  • Manipulation de contenu
  • A l'attaque

Intégration PhpBB

La plateforme de forum PhpBB permet l'utilisation simplifiée de widgets Html appelés BBCodes (le bloc de citation d'un post est un exemple de BBCode).
La page suivante documente la création de BBCodes personalisés: https://www.phpbb.com/kb/article/adding-custom-bbcodes-in-phpbb3

Exemple d'intégration pour un argument fallacieux:

PhpBB Code Usage:

[argumentDAutoriteAnonyme]{SIMPLETEXT}[/argumentDAutoriteAnonyme]

PhpBB Html Replacement:

<div class="fallacieContainer"><a class="argumentDAutoriteAnonyme" href="http://www.informationisbeautiful.net/visualizations/arguments-rhetologiques-fallacieux/" target="_blank"  title="Cliquez pour visualiser la liste des fallacies"> </a> {SIMPLETEXT}</div> <div>
