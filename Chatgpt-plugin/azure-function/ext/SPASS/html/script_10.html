10. dfg2ascii

10.1 NAME

dfg2ascii - transforms DFG files into pretty printed ASCII files

10.2 SYNOPSIS

dfg2ascii <infile>

10.3 DESCRIPTION

dfg2ascii is a program to convert a problem input file in DFG format into pretty-printed ASCII text. It prints out the axioms and the conjectures, in that order.
