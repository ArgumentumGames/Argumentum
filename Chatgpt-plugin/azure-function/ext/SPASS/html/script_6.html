6. rescmp

6.1 NAME

rescmp - tests subsumption relation between clause sets

6.2 SYNOPSIS

rescmp [ -v] file

6.3 DESCRIPTION

rescmp is a C-program that compares two clause sets specified by two DFG files. It tries to find, for each clause in the first set, a corresponding clause in the second set such that both clauses mutually subsume. It reports success only if there is a one-to-one mapping between the two sets under this condition.

6.4 OPTIONS

The following options are supported by rescmp:

-v

prints out unmapped clauses - that is clauses that have no corresponding clause - in both sets.
